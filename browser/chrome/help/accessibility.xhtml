Menggunakan Papan Kunci

Tombol Pintas Papan Kunci

Daftar tombol pintas papan kunci yang umum tersedia di Tombol Pintas Papan Kunci &brandShortName;. Informasi online dapat ditemukan di tabel perbandingan tombol pintas papan kunci yang digunakan &brandShortName;, Microsoft Internet Explorer, dan Opera.

Selalu gunakan kursor papan kunci untuk navigasi dalam halaman web

Pilihan Selalu gunakan kursor papan kunci untuk navigasi dalam halaman web memudahkan anda untuk bergerak dalam halaman web seperti halnya dalam mode penyunting read-only. Anda dapat menggunakan papan kunci untuk memilih isi pada halaman web dan menyalinnya ke clipboard. Anda dapat menggunakan fitur ini dengan melalui &pref.menuPath;, tab Utama pada panel Canggih, dan menandai &pref.singular; Selalu gunakan kursor papan kunci untuk navigasi dalam halaman web .

Anda dapat menekan tombol F7 untuk mematikan fungsi ini. Ketika anda menekan tombol F7, &brandShortName; akan menanyakan apakah Anda benar-benar ingin mematikan fitur ini. Anda dapat mematikan peringatan ini dengan menandai pilihan Jangan tampilkan dialog ini lagi.

Mulai pencarian sejak diketikkan

Mulai pencarian sejak diketikkan memudahkan anda berpindah ke teks atau tautan lain dalam halaman web. Fitur ini mempunyai dua mode penggunaan. Tekan / dan mulai mengetikkan kata yang akan dicari pada halaman saat ini atau tekan ' sebelum tulisan yang ingin dicari untuk hanya mencari tautan saja. &brandShortName; akan menerangi (highlight) teks yang cocok atau tautan yang cocok sejak Anda mulai mengetik.

Tombol pintas papan kunci berikut mengontrol fitur ini:

  • Ketik terus untuk menemukan kata yang lebih cocok. &brandShortName; akan menerangi tautan atau teks yang cocok berikutnya, atau memperingatkan anda jika tidak ada yang cocok.
  • Tekan &accelKey;+G atau F3 untuk menemukan teks atau tautan yang cocok berikutnya.
  • Tekan &accelKey;+&shiftKey;+G atau &shiftKey;+F3 untuk menemukan teks atau tautan yang cocok sebelum teks yang sedang ditampilkan atau yang muncul sebelum teks yang dicari.
  • Tekan Backspace untuk menghapus karakter terakhir yang Anda ketik pada Bar Pencarian. &brandShortName; akan mengingat teks yang sebelumnya telah diterangi.
  • Tekan Esc untuk membatalkan pencarian. Anda juga dapat membatalkan pencarian dengan memindahkan fokus, menggeser halaman dengan mouse atau papan kunci, atau menunggu saja hingga waktu habis.

Anda bisa mengubah perilaku default fitur ini &pref.menuPath;, membuka tab Umum pada panel Canggih dan menandai &pref.singular; Mulai pencarian sejak diketikkan. Jika &pref.singular; ini ditandai, Anda dapat mengetikkan secara langsung kata yang ingin dicari. Secara default pencarian dimulai untuk semua teks yang ada pada halaman. Anda masih dapat menekan ' atau / untuk mulai mencari dalam salah satu mode.

Anda juga bisa memanfaatkan fitur ini untuk mencari pada nama tombol, kotak isian, atau jenis isian form lainnya. Cari teks sebelum isian form yang diinginkan dan tekan Tab. Menekan Tab otomatis akan menghentikan pencarian dan memindahkan fokus ke isian form berikutnya, atau tombol setelah teks yang diterangi.

Huruf dan Warna

Mengganti Huruf Halaman

Beberapa situs web mungkin menampilkan teks menggunakan huruf yang susah dibaca, sedangkan situs lain mungkin menggunakan huruf default pada program browser. Anda dapat mengatur huruf default dengan membuka &pref.menuPath;, memilih panel Isi, dan memilih huruf default dan ukurannya pada bagian Huruf & Warna.

Anda dapat menentukan huruf default untuk beberapa gaya, termasuk serif, tanpa-serif (sans-serif), dan spasi tetap (monospace) dengan mengklik tombol Canggih. Walau demikian, banyak halaman web yang menentukan sendiri huruf yang akan digunakan, sehingga pilihan pengaturan Anda tidak akan berpengaruh pada halaman-halaman tersebut. Untuk mengganti huruf yang ditentukan oleh halaman seperti itu, hapus tanda pada &pref.singular; Izinkan halaman memilih hurufnya sendiri.

Perhatikan bahwa beberapa halaman web akan ditampilkan tidak terlalu baik, tergantung pilihan huruf default Anda.

Mengatur Ukuran Huruf Minimum

Beberapa situs web mungkin menampilkan teks dengan ukuran huruf yang terlalu kecil untuk dapat dibaca dengan nyaman. Untuk membuat halaman tersebut mudah dibaca, Anda dapat menentukan ukuran huruf minimum. Pada halaman web yang mencoba untuk menampilkan teks dengan ukuran yang lebih kecil dari yang telah anda atur, &brandShortName; akan memperbesar ukuran teks sesuai dengan ukuran yang telah diatur.

Untuk mengatur ukuran huruf minimum, buka &pref.menuPath;, pilih panel Isi dan klik tombol Canggih pada bagian Huruf & Warna. Anda dapat menentukan ukuran huruf minimum pada pilihan tersebut.

Perhatikan bahwa beberapa halaman web akan ditampilkan tidak terlalu baik, tergantung ukuran huruf minimum yang anda atur.

Memperbesar Ukuran Huruf Teks

Anda dapat mengubah ukuran huruf teks pada halaman web untuk sementara melalui menu Tampilan > Ukuran Teks. Ukuran yang dipilih akan tetap diberlakukan sampai Anda menutup jendela atau tab program browser, meskipun anda telah berpindah ke situs web lainnya.

Tombol pintas papan kunci berfungsi mengontrol ukuran teks:

  • Tekan &accelKey;++ untuk memperbesar ukuran teks. Teks akan diperbesar sebesar 120% dari ukuran semula, kemudian berturut-turut 150%, 200%, 300%, 450%, 675%, 1013%, dan 1519% dari ukuran semula.
  • Tekan &accelKey;+- untuk memperkecil ukuran teks. Teks akan diperkecil sebesar 90% dari ukuran semula, kemudian berturut-turut 50%, 33%, 22%, 15%, 10%, 7%, 4%, 3%, 2%, dan 1% dari ukuran semula.
  • Tekan &accelKey;+0 untuk mengembalikan ukuran teks default yang ditentukan oleh halaman.

Perhatikan bahwa beberapa halaman web akan ditampilkan tidak terlalu baik, jika anda memperbesar atau memperkecil ukuran huruf teksnya.

Mengganti Pengaturan Warna Halaman

Beberapa situs web mungkin menampilkan teks dan warna latar belakang yang susah untuk dibaca, sedangkan situs lain mungkin menggunakan pengaturan warna default program browser. Anda dapat mengatur warna default melalui &pref.menuPath;, pilih panel Isi dan klik tombol Warna pada bagian Huruf & Warna. Anda dapat mengatur warna default untuk teks dan untuk latar belakang, beserta warna default untuk tautan yang telah atau belum dikunjungi. Anda juga dapat mengatur apakah tautan harus ditampilkan dengan garis bawah.

Akan tetapi, banyak situs web yang membuat pengaturan warna sendiri. Sehingga sehingga pengaturan anda menjadi tidak berpengaruh. Untuk mengganti warna yang ditentukan oleh halaman web, hapus tanda pada &pref.singular; Izinkan halaman memilih warna sendiri.

Menggunakan Tema dengan Tingkat Kontras Tinggi

&brandShortName; otomatis mendeteksi jika anda menggunakan tema dengan tingkat kontras yang tinggi dan menampilkan semuanya dengan skema warna dengan tingkat kontras tinggi. Hal ini menggantikan semua pengaturan program browser ataupun yang diatur oleh halaman web. Selain itu, hal ini juga berpengaruh pada antarmuka &brandShortName; (semua menu, jendela, dan kotak dialog) dan isi halaman yang sedang dikunjungi.

Untuk menggunakan Tema dengan Tingkat Kontras Tinggi, buka Start Menu > Control Panel > Accessibility Options > Display and tandai &pref.singular; Use High Contrast.

Mengontrol Kandungan Isi Web

Memblokir Jendela Pop-up

Silakan dibaca Mengontrol Pop-up untuk informasi mengenai pemblokiran jendela pop-up.

Mematikan Java Applet

Beberapa situs web menyediakan antarmuka yang kaya fitur dengan menggunakan applet Java™. Akan tetapi, beberapa pengguna yang harus menggunakan papan kunci untuk bernavigasi mungkin akan mendapatkan masalah dengan beberapa applet Java yang secara otomatis mengatur fokus dan tidak menyediakan cara untuk keluar dari applet dan pindah ke bagian halaman lainnya. Jika hal ini mengakibatkan masalah bagi Anda, anda dapat mematikan fitur Java melalui &pref.menuPath;, memilih panel Isi and hapus tanda pada &pref.singular; Aktifkan Java.

Membatasi Perilaku JavaScript

Anda dapat mematikan total fungsi JavaScript melalui &pref.menuPath;, memilih panel Isi, dan menghapus tanda pada pilihan Aktifkan JavaScript. Beberapa halaman web bergantung pada fitur JavaScript ini dan mungkin tidak dapat berjalan dengan baik jika fitur JavaScript dimatikan.

Jika anda tidak menginginkan untuk mematikan total fitur JavaScript, ada beberapa pengaturan JavaScript yang dimaksudkan untuk membatasi perilaku skrip tersebut. Segera setelah menandai pilihan Aktifkan JavaScript, klik tombol Canggih untuk membuka jendela Pengaturan JavaScript Canggih. Anda dapat mengontrol skrip apakah diizinkan untuk menggeser atau mengganti ukuran jendela, menaikkan/menurunkan jendela, mematikan atau mengganti menu konteks, menyembunyikan bar status, atau mengubah teks di bar status.
