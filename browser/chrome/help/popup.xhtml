Mengontrol Pop-up

Dokumen ini menjelaskan semua &pref.plural; yang tersedia di &brandFullName; untuk mengontrol pop-up.

Pada bab ini:
  • Apa itu Pop-up?
  • &pref.pluralCaps; Pemblokir Pop-up

Apa itu Pop-up?

Jendela pop-up, atau pop-up, adalah jendela yang muncul tanpa permisi secara otomatis tanpa Anda merasa telah membukanya. Ukuran jendela bervariasi namun biasanya tidak menutup keseluruhan layar. Sebagian pop-up membuka di atas jendela aktif &brandShortName;, sementara yang lain muncul di bawah &brandShortName; (pop-under).

&brandShortName; mengizinkan Anda untuk mengontrol pop-up maupun pop-under melalui Panel Isi di &pref.pluralCaps;. Pemblokir pop-up diaktifkan secara default, supaya Anda tidak perlu khawatir harus menghidupkannya untuk menghindari pop-up muncul di &brandShortName;.

Saat memblokir pop-up, &brandShortName; menampilkan bar informasi dan sebuah ikon  di bar status. Ketika Anda mengklik tombol &pref.pluralCaps; di bar informasi, atau ikon di bar status, sebuah menu akan ditampilkan dengan pilihan berikut ini:

  • Izinkan/Blokir pop-up dari situs ini
  • Ubah &pref.pluralCaps; Pemblokir Pop-upâ€¦
  • Jangan tampilkan pesan ini (pesan info) saat pop-up diblokir
  • (tampilkan pop-up yang terblokir)

Memblokir pop-up dapat menganggu beberapa situs web: Beberapa situs web, termasuk diantaranya situs banking online, menggunakan pop-up untuk fitur penting. Memblokir semua pop-up mematikan fitur-fitur tersebut. Untuk mengizinkan situs web tertentu untuk menggunakan pop-up dan tetap memblokir yang lain, Anda dapat menambah situs web tertentu ke dalam daftar situs yang diizinkan.

Memblokir pop-up tidak selalu berhasil: Walaupun &brandShortName; memblokir hampir semua pop-up, beberapa situs web mungkin tetap dapat menampilkan pop-up menggunakan metode yang belum diketahui, walaupun telah diatur untuk diblokir.

&pref.pluralCaps; Pemblokir Pop-up

&pref.pluralCaps; Pemblokir Pop-up terletak di panel Isi dari &pref.menuPath;.

Pada panel tersebut, Anda dapat melakukan hal-hal berikut ini:

  • Jendela Pemblokir Pop-up: Hapus tanda &pref.singular; ini untuk mematikan pemblokir pop-up.
  • Dialog pengecualian: merupakan daftar situs yang Anda ingin izinkan untuk menampilkan pop-up. Dialog ini memberikan pilihan berikut ini:
    • Izinkan: Klik untuk menambah situs web ke dalam daftar pengecualian.
    • Hapus Situs: Klik untuk mengeluarkan situs web dari dalam daftar pengecualian.
    • Hapus Semua Situs: Klik untuk mengeluarkan semua situs web dari daftar pengecualian.

Catatan: Pemblokir pop-up tidak selalu berhasil dan dapat menganggu beberapa situs web. Informasi lebih lanjut tentang pemblokir pop-up, lihat Apakah itu Pop-up.
