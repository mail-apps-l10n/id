Tentang Favorit IE dan Pengaturan IE Anda

Saat Anda pertama kali memasang &brandShortName;, pengaturan Internet Explorer anda akan diimpor, termasuk favorit, cookie, kata sandi tersimpan, dan berbagai macam data lainnya. Hal ini akan menghemat waktu Anda untuk mengatur &brandShortName; sesuai kebutuhan Anda.

Fitur-fitur &brandShortName;

  • Menjelajah dengan Tab: Daripada membuka jendela browser yang terpisah untuk siap situs yang Anda ingin kunjungi, Anda dapat membuka beberapa situs sekaligus dalam jendela yang sama, dan berpindah-pindah antar situs melalui tab. Anda dapat juga menentukan sekelompok tab sebagai Beranda Anda. Untuk informasi lebih lanjut, lihat Menjelajah dengan Tab.
  • Pengelola Jendela Pop-up: Anda dapat mengizinkan atau memblokir jendela pop-up dan pop-under.
  • Pengelola Cookie: Anda dapat mengubah apa yang dilakukan &brandShortName; saat menerima cookie.
  • Pengelola Unduhan: Mengelompokkan unduhan Anda dengan menyimpannya ke dalam sebuah jendela agar lebih teratur. Anda mempunyai akses penuh ke statistik unduhan tanpa menggunakan banyak jendela untuk setiap pengunduhan.
  • Pengelola Gambar: Anda dapat mematikan gambar dari situs web tertentu atau mematikannya untuk semua situs web. Ini berguna bila Anda ingin mengurangi waktu yang diperlukan untuk memuat situs web.

Tombol Pintas Papan Kunci

Anda akan merasakan kemiripan tombol pintas di Microsoft Internet Explorer dan &brandShortName;. Untuk daftar lengkap tombol pintas, lihat Daftar Tombol Pintas &brandShortName;.
