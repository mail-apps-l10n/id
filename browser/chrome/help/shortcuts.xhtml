Tombol Pintas &brandFullName;

Berikut ini adalah daftar sebagian besar tombol pintas umum pada &brandFullName;. Apabila Anda mengaktifkan tombol pintas pengubahan teks dengan gaya Emacs di GNOME, tombol pintas tersebut juga akan bekerja pada &brandShortName;. Bila terjadi konflik antara tombol pintas penyunting teks Emacs dengan tombol pintas default (seperti halnya &accelKey;+K), tombol pintas Emacs akan mendapat prioritas lebih tinggi bila fokus berada di dalam sebuah kotak teks (termasuk bar lokasi dan bar pencarian). Pada kasus demikian Anda sebaiknya menggunakan tombol pintas alternatif bila memang didaftarkan di bawah ini.
