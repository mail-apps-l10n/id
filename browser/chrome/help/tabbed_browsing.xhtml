Penjelajahan dengan Tab

Penjelajahan dengan Tab mengizinkan Anda membuka tab, yang masing-masing menampilkan sebuah halaman web, dalam satu jendela &brandShortName;. Anda dapat membuka tautan di tab baru, tempat tautan akan dimuat ketika Anda sedang membaca halaman awal. Anda tidak harus menggunakan tab, namun bila Anda menggunakannya Anda, mungkin Anda akan berpendapat bahwa menjelajah dengan tab adalah cara yang lebih baik dan lebih cepat untuk menjelajah Web.

Menciptakan dan Memuat Tab

Untuk membuka tab baru, tekan &accelKey;+T, pilih Berkas > Tab Baru, atau klik ganda pada ruang kosong di bar Tab.

Untuk memuat tautan di dalam tab, Anda dapat melakukan salah satu hal berikut:

  • Klik tautan dengan tombol tengah. (Bila Anda memiliki mouse dengan roda di tengah, mengklik rodanya sama dengan dengan mengklik tombol tengah).
  • Seret tautan dan letakkan pada ruang kosong di bar Tab. (Apabila hanya satu halaman web yang terbuka, bar Tab mungkin tersembunyi. Lihat Mengatur Penjelajahan dengan Tab untuk informasi tentang cara mengubahnya).
  • Seret dan letakkan tautan ke sebuah tab untuk membuka tautan di tab tersebut.
  • Klik kananTekan &ctrlKey;, klik pada tautan dan pilih Buka Tautan di Tab Baru dari menu konteks.

Untuk membuka sebuah URL di Bar Lokasi di tab baru, masukkan URL-nya dan tekan &altKey;+&enterKey;.

Memindahkan Tab di dalam Jendela

Tab ditampilkan sesuai urutan waktu dibuka, yang mungkin tidak selalu Anda inginkan. Untuk memindahkan tab ke lokasi lain di dalam satu jendela &brandShortName;, cukup geser tab ke lokasi baru dengan menggunakan mouse Anda. Saat Anda menggeser tab tersebut, &brandShortName; menampilkan indikator kecil untuk menunjukkan di mana tab tersebut akan dipindahkan. Cara lainnya adalah Anda dapat menggunakan tombol pintas untuk memindahkan tab di dalam jendela bila diinginkan.

Menutup Tab

Untuk menutup tab yang sedang aktif, tekan &accelKey;+W, klik tombol Tutup Tab, klik tombol tengah pada tab, atau pilih Berkas > Tutup Tab. Untuk menutup semua tab kecuali yang sedang aktif, klik kanantekan &ctrlKey;, klik pada tab tersebut dan pilih Tutup Tab Lainnya.

Mengatur Penjelajahan dengan Tab

Untuk mengubah &pref.plural; menjelajah dengan tab, pilih &pref.menuPath; dan buka panel Tab.

Tips dan Trik

Bookmark dan Penjelajahan dengan Tab
Untuk membuat bookmark sekelompok tab:
  1. Tekan &accelKey;+&shiftKey;+D atau pilih Pilih Bookmark > Buat Bookmark Semua Tab untuk menambahkan semua tab di jendela aktif menjadi sebuah folder bookmark baru.
  2. Untuk membuka bookmark pada tab, buka menu folder bookmark dan pilih Buka di Tab baru. Anda dapat juga mengklik tengah pada folder untuk membuka isinya pada tab. Bookmark tersebut akan terbuka di tab yang terbuka, bukan di tab baru.
Beranda sebagai Tab
Daripada hanya menggunakan satu situs web sebagai Beranda, Anda dapat menjadikan sekelompok tab sebagai Beranda Anda. Pilih situs-situs web favorit Anda dan buka bersamaan pada tab-tab dalam satu jendela. Pilih &pref.menuPath; dan buka panel Utama. Di bawah Memulai, tekan tombol Gunakan Halaman sekarang. Sekarang setiap kali Anda menekan tombol Rumah, situs-situs web favorit Anda akan dimuat dengan hanya satu klik.
Menambah Tombol Tab Baru di Toolbar
Anda dapat menambah tombol Tab Baru di toolbar untuk membuka tab baru dengan mudah, dengan Mengkomposisi Item Toolbar.
Tombol pintas papan kunci
&brandShortName; memiliki sekelompok tombol pintas yang berhubungan dengan penjelajahan dengan tab; lihat daftar lengkapnya di Tombol Pintas Papan Kunci.
