Daftar istilah ini disediakan untuk informasi secara terbatas dan tidak dimaksudkan untuk dijadikan sebagai penjelasan otoritatif/definitf dan lengkap untuk istilah yang didefinisikan di bawah ini atau atas sebab akibat privasi dan/atau keamanan dari teknologi yang dipaparkan.

Daftar Istilah

Bar Lokasi
Kolom isian (dan tombol-tombol di sekitarnya) di dekat bagian atas jendela &brandShortName; tempat Anda dapat mengetikkan sebuah URL atau kata pencarian.
Bar Status
Toolbar yang muncul di bagian bawah jendela &brandShortName; yang menampilkan ikon status di sebelah kanan bawah.
beranda
Halaman yang ditampilkan setiap kali Anda memulai &brandShortName; atau ketika anda mengklik tombol Beranda. Beranda (Home) juga diartikan sebagai halaman utama dari suatu situs web di mana Anda dapat memulai mengeksplorasi situs tersebut.
bookmark
Penyimpan alamat halaman web (URL) sehingga Anda dapat dengan mudah mengunjungi alamat halaman web dengan mengklik ikon bookmark di Toolbar Bookmark atau memilih nama bookmark pada menu Bookmark.
Bookmark Hidup
Jenis bookmark khusus mirip yang berfungsi sebagai folder yang berisi tautan di dalam sebuah feed. Anda dapat menciptakan Bookmark Hidup dengan mengunjungi situs yang memiliki feed, mengklik ikon Bookmark Hidup di Bar Lokasi, dan memilih pembaca feed mana yang ingin Anda pakai.
cache
Sebuah koleksi salinan halaman web yang disimpan di hard disk komputer Anda atau di random-access memory (RAM). &brandShortName; mengumpulkan salinan ini selama Anda menjelajah Web. Ketika anda mengklik tautan atau mengetik sebuah URL untuk mengambil halaman web tertentu yang salinannya telah tersimpan di cache, &brandShortName; akan membandingkan salinan cache dengan aslinya. Apabila tidak ada perubahan, &brandShortName; akan menggunakan salinan cache tanpa mengambil ulang versi aslinya untuk menghemat waktu pemrosesan dan pengambilan.
client
Perangkat lunak (seperti halnya browser Internet) yang mengirim permintaan dan menerima informasi dari sebuah server, yang biasanya berjalan pada komputer yang berbeda. Komputer di mana perangkat lunak client dijalankan juga dapat disebut sebagai client.
cookie
Informasi kecil yang disimpan dalam komputer Anda oleh beberapa situs web. Ketika Anda mengunjungi situs tersebut, situs tersebut meminta &brandShortName; untuk meletakkan satu atau lebih cookie di hard disk Anda. Selanjutnya, jika Anda kembali mengunjungi situs tersebut, &brandShortName; akan mengirimkan balik cookie milik situs tersebut. Cookie membantu situs web melacak informasi tentang Anda, seperti isi keranjang belanja Anda. Anda dapat menentukan pengaturan cookie, bagaimana cara cookie digunakan dan berapa banyak informasi situs web yang Anda izinkan untuk disimpan pada mereka.
dekripsi
Proses mengembalikan data acak yang telah dienkripsi. Lihat juga enkripsi.
enkripsi
Proses mengacak informasi sedemikian hingga isinya dilindungi. Sebagai contoh, sambungan yang terenkripsi antara dua komputer menyulitkan pihak ketiga untuk mengembalikan atau mendekripsi informasi yang ditransfer melalui sambungan tersebut. Informasi terenkripsi dapat didekripsi hanya oleh seseorang yang memiliki kunci yang sesuai.
Extensible Markup Language (XML)
Standar terbuka untuk mendeskripsikan data. Tidak seperti HTML, XML mengizinkan pengembang situs web untuk mendefinisikan tag khusus. Untuk informasi lebih lanjut, lihat dokumen online W3C Extensible Markup Language (XML).
feed
Halaman web dalam format XML yang mengandung daftar tautan ke halaman web lainnya. Program khusus dapat membaca feed untuk membuat daftar judul berita utama dari sebuah tautan, dan secara otomatis memperbarui daftarnya ketika daftar tersebut berubah. Situs berita menggunakan feed untuk menerbitkan berita utama terkini dengan cepat, dan jurnal pribadi online seringkali menggunakan feed untuk memberitahukan pengunjung tentang entri baru dengan segera. Lihat juga Bookmark Hidup.
File Transfer Protocol (FTP)
Standar yang mengizinkan pengguna untuk mengirim berkas dari satu komputer ke komputer lain melalui jaringan. Anda dapat menggunakan &brandShortName; untuk mengambil berkas dengan FTP.
FIPS PUBS 140-1
Federal Information Processing Standards Publications (FIPS PUBS) 140-1 adalah sebuah standar pemerintah Amerika Serikat untuk implementasi modul kriptografi — perangkat keras ataupun lunak yang mengenkripsi dan mendekripsi data atau melakukan operasi kriptografi lainnya (seperti menciptakan atau memverifikasi tanda tangan dijital). Banyak produk yang dijual ke pemerintah Amerika Serikat harus memenuhi satu atau lebih standar FIPS.
halaman web
Sebuah dokumen di World Wide Web yang ditentukan oleh alamat URL yang unik. Halaman web dapat mengandung teks, tautan, dan grafik.
Hypertext Markup Language (HTML)
Format dokumen yang digunakan untuk halaman web. Standar HTML mendefinisikan tag, atau kode, yang digunakan untuk mendefinisikan layout teks, huruf, gaya, gambar, dan elemen-elemen lain yang membentuk sebuah halaman web.
ID dijital
Lihat sertifikat.
Internet
Jaringan global yang terdiri dari jutaan komputer yang saling berkomunikasi menggunakan protokol-protokol standar seperti TCP/IP. Pada awalnya dikembangkan oleh militer Amerika Serikat pada tahun 1969. Internet kemudian tumbuh dengan menyertakan institusi pendidikan dan riset, dan di akhir 1990an, menyusul jutaan perusahaan, organisasi, dan individu. Saat ini, Internet digunakan untuk mengirim email, menjelajah World Wide Web, pesan instan, milis (mailing list), dan usenet, dan banyak kegunaan lainnya.
Internet protocol address (alamat IP)
Alamat dari sebuah komputer di jaringan TCP/IP. Setiap komputer di Internet memiliki sebuah alamat IP. Client memiliki alamat IP permanen, atau alamat yang diberikan secara dinamis setiap kali terhubung ke jaringan.
Java
Bahasa pemrograman yang dikembangkan oleh Sun Microsystems. Sebuah program Java dapat berjalan di banyak macam komputer yang berbeda, sehingga menghindari keharusan programmer untuk menciptakan versi yang terpisah untuk setiap program untuk setiap jenis komputer. Setelah Java terpasang, &brandShortName; dan secara otomatis mengunduh dan menjalankan program Java (juga disebut applet).
JavaScript
Bahasa skrip yang umum digunakan untuk membangun halaman web. Programmer menggunakan JavaScript untuk membuat halaman web lebih interaktif; JavaScript seringkali digunakan untuk secara dinamis memvalidasi form dan tombol pemilih. JavaScript dapat digunakan bersama dengan Java, namun secara teknis adalah bahasa berbeda. Java tidak diperlukan agar JavaScript dapat berjalan dengan benar.
kata sandi utama (master password)
Kata sandi utama yang digunakan untuk melindungi kata sandi tersimpan dan data pribadi lainnya. &brandShortName; akan menanyakan kata sandi utama Anda ketika Anda akan mengakses data-data tersebut. Apabila Anda memiliki lebih dari satu perangkat keamanan (security device), setiap perangkat keamanan akan membutuhkan kata sandi utama yang terpisah.
kriptografi
Seni dan praktek mengacak (enkripsi) informasi dan mengembalikannya kembali (dekripsi). Sebagai contoh, teknik kriptografi digunakan untuk mengacak dan mengembalikan informasi antara situs web komersil dan &brandShortName;.
kunci pribadi
Sepasang kunci yang digunakan untuk kriptografi kunci publik. Kunci pribadi dirahasiakan dan digunakan untuk mendekripsi data yang telah dienkripsi dengan kunci publik yang bersesuaian.
mesin pencari
Program berbasis web yang mengizinkan pengguna untuk mencari dan mendapatkan informasi tertentu dari World Wide Web. Mesin pencari dapat mencari seluruh teks pada dokumen web atau dari daftar kata kunci. Mesin ini juga dapat menggunakan petugas perpustakaan yang akan meninjau dokumen web dan mengindeksnya secara manual. Umumnya, pengguna dapat mengetikkan sebuah kata atau lebih ke dalam kotak pencarian, dan mesin pencari menampilkan tautan ke halaman-halaman web yang relevan.
otentikasi
Penggunaan kata sandi, sertifikat, nomor PIN (Personal Identification Number), atau informasi lainnya untuk memvalidasi identitas melalui jaringan komputer.
Pengelola Kata Sandi
Fitur &brandShortName; yang membantu Anda mengingat sebagian atau semua pasangan nama dan kata sandi dengan cara menyimpannya di dalam hard disk komputer Anda dan memasukkannya secara otomatis ke dalam halaman ketika Anda mengunjungi situs yang membutuhkan pasangan nama dan kata sandi tersebut.
PKCS #11
Standar kriptografi kunci publik yang mengatur alat keamanan seperti smart card.
Plugin
Plugin menambah kemampuan baru ke &brandShortName;, seperti kemampuan untuk memainkan klip suara atau video. Tidak seperti aplikasi pembantu lain, sebuah plugin akan memasang dirinya sendiri ke dalam direktori Plugin di dalam direktori instalasi &brandShortName; dan biasanya dapat dibuka di dalam &brandShortName; sendiri (secara internal). Sebagai contoh, sebuah Plugin audio mengizinkan Anda mendengarkan berkas audio di sebuah halaman web atau di dalam pesan email. Contoh aplikasi plugin adalah Macromedia Flash Player dan Java.
proxy
Program perantara yang berperilaku sebagai server maupun client untuk tujuan mewakili permintaan kepada server dari client-client lain.
Secure Sockets Layer (SSL)
Protokol yang mengizinkan otentikasi bersama antara client dan server untuk tujuan membangun sambungan terotentikasi dan terenkripsi. SSL berjalan di atas tingkat protokol TCP/IP dan di bawah tingkat protokol HTTP, LDAP, IMAP, NNTP, dan protokol jaringan tingkat tinggi lainnya. Standar baru Internet Engineering Task Force (IETF) yang disebut Transport Layer Security (TLS) didasarkan pada SSL. Lihat juga otentikasi, enkripsi.
sertifikat
Sama dengan versi dijital dari sebuah kartu identitas. Sertifikat menentukan nama seorang individu, perusahaan, atau entitas lain. Sertifikat mengesahkan kunci publik yang disertakan bersama sertifikat apakah benar merupakan milik entitas yang berhak. Ketika Anda menandatangani sebuah pesan atau data lainnya secara dijital, tanda tangan dijital untuk pesan tersebut diciptakan dengan bantuan kunci privat yang bersesuaian dengan kunci publik pada sertifikat Anda.
server
Perangkat lunak (seperti perangkat lunak yang melayani halaman web) yang menerima permintaan dan mengirimkan informasi ke client, yang biasanya berjalan di komputer yang berbeda. Komputer yang menjadi tempat di mana server dijalankan juga disebut sebagai server.
situs aman (secure site))
Situs yang menggunakan enkripsi saat tersambung dengan &brandShortName; untuk menghindari pihak jahat di Internet untuk melihat data yang dikirimkan. Ketika Anda mengunjungi situs aman, &brandShortName; akan menampilkan sebuah ikon gembok di Bar Status dan Bar Lokasi. &brandShortName; juga menampilkan nama domain situs di Bar Status (untuk menghindari situs jahat mencuri data Anda) dan mewarnai latar belakang Bar Lokasi dengan warna kuning.
Sebagian situs dapat mengandung data aman dan data yang tidak aman; data yang tidak aman biasanya tidak bersifat pribadi, walaupun hal ini tidak dijamin. Untuk situs demikian &brandShortName; akan menampilkan sebuah ikon gembok yang dicoret di Bar Status dan Bar Lokasi dan tidak akan menampilkan nama domain situs tersebut di Bar Status agar anda tahu bahwa halaman web ini tidak sepenuhnya aman.
situs web
Sekelompok halaman web yang terkait oleh tautan dan dikelola oleh sebuah perusahaan, organisasi, atau individu. Situs web dapat terdiri dari teks, grafik, audio, dan berkas video, dan tautan ke situs web lainnya.
Transmission Control Protocol/Internet Protocol (TCP/IP)
Sebuah protokol Unix yang digunakan untuk menyambungkan komputer-komputer yang menjalankan berbagai macam sistem operasi. TCP/IP adalah protokol Internet yang esensial dan telah menjadi standar global.
TLS
Lihat Secure Sockets Layer (SSL).
Toolbar Bookmark
Toolbar yang dapat diubah kesukaannya yang tampil di bawah Bar Lokasi secara default di &brandShortName;. Di situ terdapat tombol-tombol untuk bookmark favorit Anda (atau folder berisi sekelompok bookmark) yang dapat Anda tambah atau kurangi.
Toolbar Navigasi
Toolbar di dekat bagian atas jendela &brandShortName; yang berisi tombol Mundur dan Maju.
Uniform Resource Locator (URL)
Alamat yang distandarkan yang memberitahukan &brandShortName; bagaimana cara menemukan sebuah berkas atau sumber daya lainnya di Web (contoh, http://www.mozilla.org). Ketik URL ke dalam Bar Lokasi &brandShortName; untuk mengakses halaman web. URL juga digunakan pada tautan di halaman web untuk berpindah ke halaman web lainnya. URL juga dikenal sebagai alamat Internet atau alamat Web.
World Wide Web
Juga dikenal sebagai Web. Sebagian dari Internet dibentuk oleh halaman-halaman web yang disimpan oleh server web dan ditampilkan oleh client yang disebut web browser (seperti halnya &brandShortName;).
